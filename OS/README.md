## 1강 운영체제 개요
[전체 내용 보기](https://jeong9216.tistory.com/297)
- 운영체제란?
  - 컴퓨터 시스템을 편리하게 사용할 수 있는 환경을 제공
  - 컴퓨터 시스템의 자원을 효율적으로 관리
- 운영체제 작업
  - CPU 스케줄링
  - 메모리 관리
  - 디스크 스케줄링

## 2강 컴퓨터 시스템 구조

- 컴퓨터시스템 구조

  - Memory
  - Mode Bit
  - Register
  - Timer
  - I/O Device
  - Device Controller
  - DMA Controller
- Interrupt
  - Inturrupt 분류
  - Inturrupt 제어권

- 프로그램 실행(Memory Load)
